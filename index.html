Activity Generator

It connects to an external API to provide diverse activity suggestions, from hobbies to tasks, ensuring users always have something new to try. The application is an excellent showcase of my ability to work with APIs and create intuitive, engaging user interfaces, providing practical solutions through technology in difficult times.

Cinematic Explorer

Cinematic Explorer is a robust React app that allows users to search for movies, TV shows, or episodes by title. Utilizing a movie API, it displays detailed information including a poster image, title, description, release date, and a rating out of 10. The app even provides visual feedback with thumbs up or thumbs down icons based on the rating. Additionally, it stores the last three searches for quick access, showcasing my proficiency in React and API integration to build highly interactive and useful applications.

Cocktail Disco

This React-based web application caters to cocktail enthusiasts by providing a sleek, user-friendly interface to explore a wide variety of cocktails. Each search triggers an API call to display curated cocktail recipes, complete with ingredients and preparation steps. The app also stores and displays users’ last three searches, enhancing usability and engagement. The combination of API integration and smooth React development makes this project a perfect example of my ability to create polished, feature-rich web experiences.

Covid Workout Roulette

Covid Workout Roulette is an innovative web app developed with HTML, CSS, and JavaScript, aimed at keeping users active during the pandemic. It pulls random workout videos from YouTube based on the user’s preferences for activity type (yoga, strength, or cardio) or workout duration. This app provides an engaging way to discover new workouts while demonstrating my ability to integrate external APIs, design engaging interfaces, and build applications that cater to user needs during challenging times.
